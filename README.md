## How To API

```sh
Terminal 1 
$ npm run start
```

## How To Start Chatbot

```sh
Terminal 2
$ node bot.js
```

### Bot Commands

| Command | Function |
| ------ | ------ |
| /start | Start Chat Bot |
| /product | Get Product List |
| /checkcart | Check Cart Order |
